- aaWallet钱包地址
- erc20地址: 0x08F8BE95b2F5d3EE18Ec6FF2bb25ef8351bFd637
- platform地址: 0x3Bbb3AC09F662Af8BAD15983F5A1E4de79f4f059

- zeroDev的Gas代理策略
